# AI-World 上线运营清单

更新时间：2026-03-17

本清单只适用于当前单服务器生产发布模型。

## 1. 发布前

- [ ] 确认本次上线 commit/tag 已固定，不再继续扩需求
- [ ] 本地通过 `npm exec tsc --noEmit`
- [ ] 本地通过前端构建 `npm run build`
- [ ] 后端关键测试通过
- [ ] `npm run release:preflight:production` 通过
- [ ] `server/.env.production` 已显式关闭 demo / public sample invite
- [ ] `SESSION_SECRET` 已替换为强随机值
- [ ] 已确认 Assistant / Knowledge Base 本次是否开放，并同步前后端开关
- [ ] 已完成数据库备份

## 2. 生产部署

- [ ] 同步当前代码到服务器 `/opt/aiworld`
- [ ] 在服务器构建或准备好本次镜像
- [ ] 执行 `REGISTRY=<registry> IMAGE_TAG=<tag> ./scripts/deploy-stack.sh`
- [ ] 执行 `./scripts/check-stack.sh`
- [ ] 执行 `./scripts/verify-observability.sh`

## 3. 发布后自动/脚本验收

- [ ] `https://ai-world.asia/health` 返回 `200`
- [ ] `https://ai-world.asia/ready` 返回 `200`
- [ ] `https://ai-world.asia/api/auth/invite/samples` 不再返回公开邀请码
- [ ] `npm run test:live:prod-smoke` 通过

## 4. 发布后人工验收

- [ ] 登录页正常
- [ ] 注册邀请码流程正常
- [ ] 发布页正常
- [ ] Admin Review 正常
- [ ] Enterprise Dashboard 正常
- [ ] WebSocket/消息页基础可用

## 5. 发布封板

- [ ] 停掉服务器上遗留的 staging 容器与配置
- [ ] 记录实际发布 tag、发布时间、验证结果
- [ ] 更新 `docs/项目审计与剩余任务.md`
- [ ] 更新本次发布记录

## 6. 上线后 24 小时观察

- [ ] Nginx 错误日志无持续异常
- [ ] API 容器无持续重启
- [ ] PostgreSQL / Redis 无异常资源告警
- [ ] 注册、登录、发布、申请主链路无用户反馈故障
